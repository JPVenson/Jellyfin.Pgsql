PostgreSQL Database Configuration

This plugin uses environment variables for configuration. Set the following environment variables before starting Jellyfin:

Required Environment Variables

  • POSTGRES_PASSWORD - Database password (required)

Optional Environment Variables (with defaults)

  • POSTGRES_HOST - Database host (default: localhost)
  • POSTGRES_PORT - Database port (default: 5432)
  • POSTGRES_DB - Database name (default: jellyfin)
  • POSTGRES_USER - Database user (default: jellyfin)

Example Docker Compose

environment:
  - POSTGRES_HOST=postgres
  - POSTGRES_PORT=5432
  - POSTGRES_DB=jellyfin
  - POSTGRES_USER=jellyfin
  - POSTGRES_PASSWORD=your_secure_password
